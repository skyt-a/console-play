# console-play

This library provides the function to display decorated text(Ascii Art) in the browser console

## Quick Start
Install:
```
npm install console-play
```

Simple usage:

```ts
import { consolePlay } from "console-play";

consolePlay({
    text: "Hello, World!",
    options: {
        styleOptions: {
            color: "red",
        },
    },
});
```
